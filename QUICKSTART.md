# VIX Fin - Instruções Rápidas

## 🚀 Começar Agora

### 1. Instalar
```bash
cd "VIX Fin"
pip install -r requirements.txt
```

### 2. Executar
```bash
cd backend
python app.py
```

### 3. Abrir
Navegador: **http://localhost:5000**

---

## 🧪 Testar API

```bash
# Em outro terminal, na pasta VIX Fin:
pip install requests
python test_api.py
```
