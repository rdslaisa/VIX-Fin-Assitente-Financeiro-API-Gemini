## Sobre📄

O VIX Fin é uma plataforma web educacional voltada para quem está começando a investir. O objetivo é tornar o mercado financeiro mais acessível, com linguagem simples e ferramentas práticas.
O projeto está em desenvolvimento ativo, com planos de evoluir para uma aplicação web completa com interface no browser.

## Funcionalidades🔧

- 📚 **Educação Financeira** — conceitos e tipos de investimento explicados de forma didática
- 📈 **Cotações em Tempo Real** — ações brasileiras, líderes globais, moedas e criptomoedas
- 🧮 **Simulador** — projete o crescimento do seu patrimônio com diferentes cenários
- ✨ **Recomendações Personalizadas** — sugestões baseadas no seu perfil e objetivos
- 🙋🏻‍♀️ **Atena** — assistente virtual com IA para tirar dúvidas sobre educação financeira

## Tecnologias💻 

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **Dados de mercado:** yfinance
- **IA:** Google Gemini

## Aviso⚠️

O VIX Fin é uma plataforma **exclusivamente educacional**. Nenhuma informação aqui deve ser interpretada como recomendação de compra ou venda de ativos. Decisões de investimento devem ser tomadas com apoio de um profissional certificado (CVM/ANBIMA).
